# odin-recipies
### Learning objectives
*  create a repo in github and clone it to a local folder
* set up a README
* demonstrate ability to create html pages and link them using relative paths
* render images to a web page
* use ordered and unordered lists
* publish to the web for public viewing

### Bugs
* W3schools validation showed 2 errors, where the alt tag had been used on anchor tags and h3 tags had been used after h1, skipping h2.
* Resolved by replacing alt tags with title tags and h3 tags with h2 tags
* All html pages passed validation after these corrections
